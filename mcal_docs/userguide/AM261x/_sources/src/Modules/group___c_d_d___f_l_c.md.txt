### function Cdd_Flc_ConfigureRegion

```
Std_ReturnType Cdd_Flc_ConfigureRegion(
    Cdd_Flc_HwUnitType HwUnitId,
    Cdd_Flc_RegionId RegionId,
    const Cdd_Flc_RegionConfigType * RegionCfgPtr
)
```


**Brief**: 
Service to configure a FLC region. 

**Parameters**: 

  
* **HwUnitId** HW ID. This param should be less than CDD_FLC_HW_UNIT_MAX 
  
* **RegionId** Region ID to configure. This param should be less than CDD_FLC_REGION_ID_MAX 
  
* **RegionCfgPtr** Pointer to the region configuration parameter. This should not be NULL.


**Return**: If the configuration is successful, returns E_OK else returns E_NOT_OK 

**Note**: If the requested region is already in enabled state, the API returns E_NOT_OK. It is expected that the caller first calls Cdd_Flc_DisableRegion() API before calling this API in this scenario.

### function Cdd_Flc_EnableRegion

```
void Cdd_Flc_EnableRegion(
    Cdd_Flc_HwUnitType HwUnitId,
    Cdd_Flc_RegionId RegionId,
    boolean EnableWithoutCopy
)
```


**Brief**: 
Service to enable and start the FLC copy from source to destination. 

**Parameters**: 

  
* **HwUnitId** HW ID. This param should be less than CDD_FLC_HW_UNIT_MAX 
  
* **RegionId** Region ID to start. This param should be less than CDD_FLC_REGION_ID_MAX 
  
* **EnableWithoutCopy** Set this to TRUE when the region has to be enabled without a copy 


### function Cdd_Flc_IsRegionCopyDone

```
boolean Cdd_Flc_IsRegionCopyDone(
    Cdd_Flc_HwUnitType HwUnitId,
    Cdd_Flc_RegionId RegionId
)
```


**Brief**: 
Service to check if the initiated FLC copy is completed. 

**Parameters**: 

  
* **HwUnitId** HW ID. This param should be less than CDD_FLC_HW_UNIT_MAX 
  
* **RegionId** Region ID to check. This param should be less than CDD_FLC_REGION_ID_MAX


**Return**: TRUE if copy is completed. FALSE otherwise.
