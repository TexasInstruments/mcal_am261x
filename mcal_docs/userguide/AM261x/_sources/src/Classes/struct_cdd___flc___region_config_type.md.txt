# Cdd_Flc_RegionConfigType



FLC Region Config type. 


`#include <Cdd_Flc.h>`

## Structure Members

|  Member Type   | Member name    | Brief |
| -------------- | -------------- | ----- |
|  uint32 | srcStartAddr |  Source start address. Data at this address is copied. This address should be aligned to 4KB boundary.  |
|  uint32 | srcEndAddr |  Source end address, upto this address data is copied. This address should be aligned to 4KB boundary.  |
|  uint32 | destStartAddr |  Destination start address, this address is also written. This address should be aligned to 4KB boundary.  |
