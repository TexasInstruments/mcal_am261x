    all: misc: add HTML to JSON converter for test results
    
    Add test_result_html_to_json.py script to convert MCAL test automation
    HTML reports to JSON format for easier programmatic processing.
    
    Features:
    - Parses test automation HTML reports and extracts structured data
    - Outputs JSON with metadata, summary, repo revisions, component
      summary, and test case details
    - Excludes test_engine_summary section from output
    - Extracts only JIRA ID from test case links (removes URL)
    - Removes log_http, board_name, and assignee fields from test cases
    - Handles malformed HTML with duplicate closing tags gracefully
    - Uses only Python standard library (no external dependencies)
    - Supports Python 3.6+ (f-strings and typing)
    - Add test result comparison script with test suite
    - Test input files stored in test/inputs/ folder.
    
    Usage:
      python test_result_html_to_json.py results.html results.json
    
    Also adds requirements.txt documenting the standard library dependencies.
    
    Fixes: MCAL-35884
    
    Signed-off-by: Sivaraj R <sivaraj@ti.com>
